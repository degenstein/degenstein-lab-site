Crypto-backed loans (feature overview)

CoinDepo may offer crypto-backed loans, which can let users borrow against collateral without selling. This can be useful for liquidity, but it introduces loan terms + liquidation risk if collateral value drops.

  • Verify supported collateral assets, LTV, rates/fees, and liquidation rules in-app.
